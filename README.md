## Run Locally :-
To run Tourism-Website locally, run this command on your git bash:

**Linux and macOS:**
``` bash  
sudo git clone https://github.com/Kaif0412/Tourism-Website
```
**Windows:**
``` bash  
git clone https://github.com/Kaif0412/Tourism-Website
```
